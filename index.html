CCMS Performance Portal

This site provides visualisations of data collected as part of the operational management of the LAA's Client & Cost Management System. The collection and/or collation of much of the data requires manual intervention. The site should therefore not be mistaken for a realtime monitoring tool.

Performance metrics

  • Batch Schedule
  • Business Rules Monitor
  • Database Size
  • Deferred Upload
  • Network Latency
  • Network Throughput
  • Purging
  • User Locations (UK)
  • User Locations (World)

Unless otherwise specified, the underlying data is maintained in Google Sheets.

Service Status

The Service Status application is not based on realtime data and — as such — may not be an accurate reflection of the current status of the service.
