About the data

This page shows data collected using the Oracle E-Business Suite Network Test Form. These are not real-time data, but data collected manually. As such it may not reflect the current network performance.

These data are held in — and can be edited using — Google Sheets.

What does it measure?

The network throughput figures relate to the communications between the Forms Server and the Java client on the end user's device. Three network routes are measured:

  1. VDI - the Java client is inside the 6dg network.
  2. Internet - the end user device is connected directly to the internet.
  3. Dom1 - the end user device is connected to the internet via the Dom1 network.

Why does it matter?

Users of Oracle Forms can experience performance issues when accessing Forms via a low throughput network.
